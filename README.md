# marimo-quarto

Documents that specify the `marimo/quarto` extension e.g.

```yml
---
title: Marimo Tutorial
format: html
filters:
  - dmadisetti/quarto-marimo/marimo/quarto
---
```

call the relevant extension in `_extension/marimo/quarto`.
